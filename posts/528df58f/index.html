Reinventing the Wheel: JSON Conversion Library (Ⅲ)

Posted on 2019-04-17 In algorithm

Building a Java library which implements JSON conversion between JSON strings and Java beans. The source code is available in here.

Object => JSONElement

Firstly, check the type of the object to see if it can be directly converted to JSONPrimitive, JSONList, JSONMap. Otherwise, treat it as a record and use reflection to extract the field name and value and convert it to JSONMap.

JSONElement => Object

The basic idea is to get the field name of the target type and then take the value into recursion.

The source code is too long to show it here. Check ObjectAnalyzer instead.
